## 引入依赖
```
<dependency>
    <groupId>org.elasticsearch.client</groupId>
    <artifactId>transport</artifactId>
    <version>7.6.2</version>
</dependency>
```

### 基本概念

* 索引（indices）
* 类型（type）
* 文档（Document）
* 字段（Field）
* 映射配置（mappings）

### 单一文件API
* Index API
* Get API
* Exists API
* Delete API
* Update API
* Term Vectors API

### 多文档API
* Bulk API
* Multi-Get API
* Reindex API
* Update By Query API
* Delete By Query API
* Rethrottle API
* Multi Term Vectors API
